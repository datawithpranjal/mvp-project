# The Data Foundry Product Roadmap

The Data Foundry is a practice-first platform for Data Engineering interviews, production scenarios, project simulation, and job readiness.

## Phase 1: Scenario Playground

- Production-style SQL/debug/design/code-review scenarios.
- Free and premium cards visible in the public library.
- Hints, attempt history, draft saving, model answer reveal, self-rating, and completion tracking.
- Local progress in MVP; backend persistence later.

## Phase 2: AI Evaluator

- Provider-agnostic evaluator interface.
- Mock deterministic evaluation when no AI key is configured.
- Scoring dimensions: problem understanding, root-cause thinking, solution design, trade-offs, monitoring/testing, and interview clarity.
- Do not leak locked premium model answers through AI feedback.

## Phase 3: Project Simulator

- Visual, decision-based project simulations without real Spark/Kafka/Airflow infra.
- First simulator: E-commerce Orders Data Pipeline.
- Missions cover raw events, partitioning, deduplication, late records, gold marts, Airflow green-but-wrong-data, and dashboard mismatch.

## Phase 4: SQL/PySpark Labs

- Dedicated daily drills for SQL and PySpark.
- More output-matching SQL validators.
- PySpark reasoning labs before introducing executable Spark infra.

## Phase 5: Community And Challenges

- Weekly production incident challenges.
- Leaderboards or cohorts only if they stay professional and useful.
- Creator walkthroughs from Data with Pranjal.

## Phase 6: DA/DS/ML/AI Expansion

- Expand the platform model into Data Analytics, Data Science, Machine Learning, and AI engineering tracks.
- Keep The Data Foundry as the scalable product brand.
